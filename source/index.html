Holiday Gift Box

UAID's Holiday Gift Box impacts the lives of nearly 1,400 people with intellectual disabilities each year from all corners of the state. The program provides basic need resources like shoes, coats, sheets, disposable undergarments, and personal hygiene items. Simliar in design to programs like Angel Tree or Sub for Santa, Holiday Gift Box is the only program that supports both adults and children with disabilities.
